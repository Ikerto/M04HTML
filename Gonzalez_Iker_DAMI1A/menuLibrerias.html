Librerías

Estas son las librerías disponibles en esta página.

SNPassoc

Se implementan funciones para realizar la mayoría de los análisis comunes en los estudios de asociación del genoma.

MultiDataSet

Implementación de los MultiDataSet y ResultSet del BRGE. MultiDataSet está diseñado para integrar conjuntos de datos multiómicos y ResultSet es un contenedor para resultados ómicos.

scoreInvHap

scoreInvHap puede obtener el estado de inversión de las muestras de inversiones conocidas. scoreInvHap utiliza datos SNP como entrada y requiere información sobre la inversión.

rexposome

Paquete que permite explorar el exposoma y realizar análisis de asociación entre exposiciones y resultados en salud.

CNVassoc

Realiza análisis de variantes comunes del número de copias y polimorfismos de nucleótido único imputados en estudios basados en la población.

methylclock

Este paquete permite estimar la metilación del ADN cronológico y gestacional así como la edad biológica usando diferentes relojes de metilación.

dsML

Librería de DataSHIELD que implementa técnicas de aprendizaje automático en datos federados.

BigDataStatMeth

Métodos de álgebra basica mediante algoritmos paralelizables para tratar datos grandes de estudios ómicos.

CTDquerier

Librería para descargar y visualizar datos de Comparative Toxicogenomics Database (http://ctdbase.org/).

dsOmics

Librería de DataSHIELD para analizar datos ómicos en estudios multicéntricos.

epimutacions

Librería para detectar mutaciones epigenéticas en enfermedades raras.

MADloy

Detección de pérdida del cromosoma Y con datos genéticos.

omicRexposome

Librería de Bioconductor para integrar datos ómicos con el exposoma.

RecombClust

Detección de patrones de recombinación en datos genéticos.

RNAseq

Expresión diferencial para datos de secuenciación masiva.
